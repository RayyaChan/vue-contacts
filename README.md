# Contacts
### 简述
这是一个通讯录的project，以web方式展示，目前只支持PC端，暂无移动端的适配。

### 实现的功能

* [x] 登录
* [x] 注册
* [x] 联系人增删查改
* [x] 联系人分组
* [x] 联系人导入
* [x] 联系人导出

### 技术栈
  
* Vue
* Vuex
* VueRouter
* Axios
* Vue-cli
* ElementUI
* Node
* Express

### 构建
需要启动两个服务，一个前端，一个后台
``` bash
# install dependencies
npm install

# server in dev mode, with hot reload at localhost:12221
npm run dev

# build for production
npm run build

```

``` bash
# server with node
node server.js
```
